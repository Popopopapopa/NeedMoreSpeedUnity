# Need More Speed

## Описание проекта
**Need More Speed** — это гоночная игра, созданная на **Unity**, в которой игроку нужно соревноваться на скорости, проходить круги по трассе и показывать лучший результат.  
Проект сделан в стиле аркадных гонок с быстрыми суперкарами, таймером кругов, ботами-соперниками и меню игры.

## Описание геймплея
Игрок управляет машиной и участвует в заезде на гоночной трассе.  
Главная задача — как можно быстрее проходить круги, обгонять соперников-ботов и улучшать своё время.

В игре реализованы:
- заезды по круговой трассе;
- отсчёт времени круга;
- отображение текущего круга;
- отображение времени предыдущего круга;
- сравнение результата круга (улучшилось время или ухудшилось);
- боты-соперники;
- стартовое меню;
- пауза;
- рестарт гонки;
- выход в главное меню.

## Управление
### Управление машиной
- **W / S** — газ / тормоз
- **A / D** — поворот влево / вправо

### Дополнительное управление
- **Клавиша паузы** — поставить игру на паузу
- **Клавиша рестарта** — перезапустить гонку
- **Клавиша выхода в меню** — выйти в главное меню

### Управление камерой
Камера работает в режиме вида от третьего лица и следует за машиной.  
Поворот камеры осуществляется кнопкой мыши, выбранной в настройках скрипта.

## Инструкция по запуску
1. Открыть проект в **Unity Hub**.
2. Выбрать нужную версию Unity и открыть проект.
3. Открыть стартовую сцену через папку **Scenes**.
4. Нажать кнопку **Play** в редакторе Unity.
